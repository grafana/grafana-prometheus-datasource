# grafana-prometheus-datasource

This package exists **only for versioning purposes**.

`@changesets/cli` cannot target the workspace root, so this stub package stands
in for the root: its `version` and `CHANGELOG.md` are bumped by
`changeset version`, and `scripts/sync-changelog.js` mirrors them back to the
workspace root. There is no source code here and nothing should be added.

See `scripts/version-changeset.js` for the full flow.
